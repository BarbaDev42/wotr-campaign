# Immagini

Questa cartella contiene tutte le immagini usate nella wiki — ritratti di personaggi, PNG di NPC, illustrazioni di luoghi, oggetti, ecc.

## Convenzione nomi file

| Tipo | Formato | Esempio |
|------|---------|---------|
| Personaggio | `pg_nomepersonaggio.png` | `pg_kaelen_vortanis.png` |
| NPC | `npc_nomepersonaggio.png` | `npc_hulrun.png` |
| Luogo | `luogo_nomeposto.png` | `luogo_kenabres.png` |
| Oggetto | `oggetto_nomeoggetto.png` | `oggetto_spada_fiamma.png` |
| Fazione | `fazione_nomefazione.png` | `fazione_aquile_di_cenere.png` |

## Come referenziare nelle note

```markdown
![[pg_kaelen_vortanis.png]]
```

Obsidian e Quartz risolvono automaticamente il percorso — basta il nome del file, senza path completo.
